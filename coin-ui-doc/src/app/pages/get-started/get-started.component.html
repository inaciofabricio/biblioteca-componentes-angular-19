Qual é a proposta da Coin UI Angular?

Motivado pela curiosidade de compreender melhor o trabalho de profissionais de UX/UI, concluí a Formação Design System na Alura. Com base nos conhecimentos adquiridos nesses cursos, decidi criar uma biblioteca de componentes em Angular 19 (versão atual). Embora não seja uma biblioteca completa, ela reúne o básico e serve como inspiração para quem deseja criar sua própria biblioteca de componentes.
